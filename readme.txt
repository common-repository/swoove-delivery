Transforming e-commerce logistics in Africa

== Description ==
The plugin is to enable e-commerce webistes be able to get shipping estimates and request shipping from their store


== Installation ==
1. Upload the plugin folder to the ‘/wp-content/plugins/’ directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. Thats it – you can now configure the plugin.

More detailed instructions on how to set up your store and configure it are consolidated on on the page 
<a href="https://www.swoove.delivery/plugin-setup" target="_blank">Set Up Guide</a>

== Frequently Asked Questions ==
= After the setup, my site is still saying “There doesn‘t seem to be any available shipping methods”. =

You can enable developer mode in settings and debug messages will start showing up in cart/checkout page, from which you will get a clue what is the issue about. Hope Swoove shipping method is enabled.
